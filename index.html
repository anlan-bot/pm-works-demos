🐾 毛孩疫苗本

🔍 需求来源

竞品调研8款宠物健康App，发现记录繁琐、提醒不灵活、导出缺失；访谈12位宠物主，9人常忘疫苗日期，7人希望拍照识别疫苗标签。

✅ MVP功能清单

  • 多宠物档案（增删改）
  • 疫苗记录 + 有效期自动计算
  • 花费明细追踪
  • 离线OCR识别疫苗瓶身
  • 智能提醒（稍后/今日忽略/每日定时/按宠物筛选）
  • 数据本地化 + Excel导出 + JSON备份

🔄 迭代历史（关键版本）

  • v1.0 (2025.09)：基础增删改查，手动录入疫苗。
  • v1.1 (2025.10)：下拉选疫苗自动算有效期，修复日期bug。
  • v1.2 (2025.11)：集成Tesseract.js OCR，通过Canvas预处理将准确率从40%提至80%+。
  • v1.3 (2025.12)：新增“板块使用举例提示”。
  • v1.4 (2026.01)：按月聚合花费视图、导出Excel、按宠物筛选提醒。
  • v1.5 (2026.02)：优化提醒状态机，支持稍后提醒/今日忽略/每日定时。

💬 用户反馈摘要

🐶 “疫苗提醒很准，导出Excel备份太实用，再也不怕换手机丢数据。” —— 金毛主人·李女士

✍️ 星途·创作舱

🔍 需求来源

调研12位网文作者，普遍痛点：大纲与写作割裂、角色设定散落、灵感易丢失。竞品分析发现缺少轻量级一站式创作工具。

✅ MVP功能清单

  • 多项目管理 + 标签分类
  • Markdown大纲编辑器（支持格式化插入与实时预览）
  • 角色与世界观设定面板
  • 参考书目管理
  • 内置标题灵感库（400+词条）
  • 数据本地存储 + 导入/导出JSON备份

  • v1.0 (2025.09)：基础大纲编辑器 + 本地存储。
  • v1.1 (2025.10)：增加角色和世界观面板。
  • v1.2 (2025.11)：上线标题灵感库（400+词条）。
  • v1.3 (2025.12)：增加多主题切换（含墨水屏高对比度“墨”主题）。
  • v1.4 (2026.01)：支持Markdown大纲附件上传与预览、自定义标签/大纲类型/鼓励语管理。
  • v1.5 (2026.02)：新增章节创作记录与文件挂载、随机标题抽取与动态欢迎语系统。

📋 剧本·复盘舱

🔍 需求来源

针对编剧/策划长内容创作中「版本迭代难追踪、修改原因易遗忘、关联文件易散落」的核心痛点，竞品调研发现缺乏专门的“写后管理”工具。

✅ MVP功能清单

  • 剧本-进度-复盘三位一体管理模型
  • 进度记录（日期、标题、文档附件、复盘反思）
  • 标签分类与全局标签库
  • 文件附件（txt/md/pdf/doc等）本地存储与预览
  • 一键导入/导出完整项目备份（JSON）
  • 退出/刷新前自动备份机制

🔄 迭代历史（关键版本）

  • v1.0 (2025.10)：基础记录增删改查 + 本地存储。
  • v1.1 (2025.11)：增加文件附件上传与预览。
  • v1.2 (2025.12)：新增标签系统与标签管理。
  • v1.3 (2026.01)：内置月度回顾看板（统计记录数、活跃剧本、标签分布、高频词）。
  • v1.4 (2026.02)：支持导出Word/CSV/JSON备份，增加自动备份开关。

🎙️ 会议·反刍舱

🔍 需求来源

针对会议“记录即遗忘”、待办难追踪、决策无法有效反刍的痛点，调研10位职场人，8人表示会议纪要写完就丢，7人希望有自动提取待办和提醒功能。

✅ MVP功能清单

  • 录音/音频上传 → 浏览器端Whisper转文字
  • DeepSeek API / 本地模型 / 内置规则提取议题、待办、反刍问题
  • 待办追踪（支持负责人、截止时间、完成状态）
  • 反刍提醒（1小时/1天/自定义分钟，基于通知API）
  • 周报自动汇总已完成待办
  • 导出CSV/Markdown/Word/JSON备份

🔄 迭代历史（关键版本）

  • v1.0 (2025.11)：纯本地规则提取待办+议题，基础历史存储。
  • v1.1 (2025.12)：集成DeepSeek API，支持智能分析。
  • v1.2 (2026.01)：增加浏览器Whisper离线转写，支持多格式音频。
  • v1.3 (2026.02)：增加本地模型端点（Ollama/LM Studio）支持，内置规则作为兜底。
  • v1.4 (2026.03)：新增反刍提醒面板、周报看板、多格式导出（DOC/DOCX/CSV/MD）。

💬 用户反馈摘要

🎤 “Whisper转写准确率很高，英文会议也能识别，省去了手动整理时间。” —— 产品经理·Ryan
✅ “待办追踪和反刍提醒让我真的会回头复习会议内容，不再是记完就忘。” —— 运营·niiico
🔧 “支持本地模型很好，公司内网也能用，不用担心数据上传。” —— 技术负责人·米周
