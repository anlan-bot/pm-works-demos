## 🚀 核心项目

### 🐾 毛孩疫苗本 — AI驱动的宠物健康管理工具
**解决宠物主“疫苗/花费记录分散、到期易遗忘”的核心痛点**

- **产品策略**：竞品分析8款App + 12位宠物主访谈，定义轻量记录+智能提醒的差异化方向
- **核心功能**：多宠物档案、离线OCR疫苗识别（准确率40%→80%+）、灵活提醒系统（稍后/今日忽略/按宠物筛选）、Excel数据导出
- **数据验证**：A/B测试模拟显示，优化提醒策略后设置转化率提升61.5%，7日留存提升41.9%
- **用户声音**：“疫苗提醒很准，导出Excel备份这个功能很实用。” —— 金毛妮可主人·李女士

### ✍️ 星途·创作舱 — 网文创作者效率工具
**解决创作者“大纲与写作割裂、素材管理混乱”的痛点**

- **产品设计**：集成Markdown大纲编辑器、角色/世界观设定面板、400+标题灵感库，构建一站式创作工作台
- **体验打磨**：提供墨水屏等多主题切换、自定义标签管理，迭代10+项交互优化
- **用户声音**：“墨水屏主题好用，在电子纸上写大纲很舒服。” —— 签约作者·miki

### 📋 剧本·复盘舱 — 编剧进度与版本复盘工具
**填补长内容创作中“版本迭代难追踪、修改原因易遗忘”的工具空白**

- **创新模型**：设计“剧本-进度-复盘”三位一体结构，将修改记录与复盘原因强关联
- **核心功能**：进度追踪、全局标签库、月度回顾看板（含高频词分析）、一键导出Word/CSV备份
- **用户声音**：“每次修改剧本原因都能记录下来，再也不用回忆‘当时为什么这么改’。” —— 编剧·veka

### 🎙️ 会议·反刍舱 — AI会议复盘与行动管理
**解决会议“记录即遗忘、待办难追踪”的低效循环**

- **闭环设计**：录音 → Whisper转文字 → DeepSeek/本地模型提取待办 → 反刍提醒 → 周报自动汇总
- **技术亮点**：浏览器端运行Whisper，零服务端成本；支持导出Markdown/Word/CSV
- **用户声音**：“Whisper转写相对来说准确率还算高，待办追踪让我真的会回头复习会议内容。” —— 产品经理·Ryan
